3. [Hash_Table]
This program implements a Hash Table using chaining to handle collisions. Each key-value pair is stored in a list at an index determined by a hash function. The HashTable class provides methods to insert, retrieve, and remove elements by key, as well as a __str__ method to display the table. This allows efficient access, update, and deletion of data based on unique keys.
